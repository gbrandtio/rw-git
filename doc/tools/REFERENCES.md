# Consolidated Citation Index

The single canonical index of every academic paper, book, and standard cited
across rw-git's documentation. Per-tool documents (`doc/tools/**`) and
[`TOOLS_ACADEMIC_FOUNDATIONS.md`](../TOOLS_ACADEMIC_FOUNDATIONS.md) describe
*how* each paper is applied; this file is the authoritative list of *what* is
cited, so a citation is corrected in one place.

Entries are alphabetical by first author's surname. The **Used by** column
references section numbers in
[`TOOLS_ACADEMIC_FOUNDATIONS.md`](../TOOLS_ACADEMIC_FOUNDATIONS.md) (e.g. 4.6 =
`evaluate_comments`), where each tool's use of the paper is explained.
